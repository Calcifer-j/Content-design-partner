Build a content design partner (simple agent)

An AI agent is a software program that uses an AI model as its "brain" to pursue a goal on its own. Instead of just answering questions, an agent can break a complex problem into steps, use external tools (like searching the web, or sending an email), and check its own work. Agents include:

  • projects or custom GPTs (custom prompts with access to basic tools like web search or a private document library)
  • task-specific agents (designed for a specific workflow, with little or no human intervention)
  • multi-agent systems (where many agents work together to achieve a common goal)

The content design partner is a basic agent, which uses instructions and tools to complete a set of review tasks, and check content against other information.

The basic agent: 2i and GOV.UK style check

The basic Content design partner is a basic agent which checks content against GOV.UK style and standards, and outputs a table of potential issues, suggested changes and sources for review. I've also built extra modules that add more capabilities - each one below gives you a complete, ready-to-use version of the prompt with that capability included, so there's nothing to combine yourself.

This prompt will review content against these GOV.UK content design principles, including:

  • plain English: short sentences, everyday words, no jargon
  • active voice
  • sentence case for headings
  • numbered lists for steps that happen in order
  • bullet points for anything else

It will then share it's review findings in a table which identifies potential issues, information sources and potential solutions.
